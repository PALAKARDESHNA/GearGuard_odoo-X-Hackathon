# GearGuard Frontend

React + TypeScript frontend for GearGuard Maintenance Tracker.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

The app will run on `http://localhost:3000`

## Build

```bash
npm run build
```

## Features

- Equipment Management
- Maintenance Teams
- Kanban Board (Drag & Drop)
- Calendar View
- Reports & Analytics
